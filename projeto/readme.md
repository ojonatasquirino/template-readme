# clonagem e instalação <a name="id06"></a>

Clone este repositório usando o comando:

```bash
git clone [link do repositório remoto]
```

Instale na pasta do projeto as dependências com uso do npm ou yarn:

```bash
npm install

ou

yarn install
```

Inicie o projeto com:

```bash
npm start

ou

yarn start
```
